# Kafka-Utils — инструкция для игроков

Клиентский мод для Minecraft **1.21.11** (Fabric). Ставится только тебе, на
сервере ничего настраивать не надо. Ниже — как поставить и как пользоваться
всеми фичами.

---

## 1. Установка

1. Поставь **Fabric Loader** для версии 1.21.11 и **Fabric API**.
2. Кинь `kafkautils-*.jar` (без `-sources`) в папку `mods/`.
3. Запусти игру. При старте в логах будет строка
   `[KafkaUtils] Initialized — open the menu with X`.

---

## 2. Меню модулей (ClickGui)

- Открыть/закрыть — клавиша **`X`** (можно переназначить в
  **Options → Controls → Key.kafkautils.open_gui**).
- Модули разбиты по категориям слева (Combat, Render, Friends).
- Клик по модулю — вкл/выкл. Стрелка/раскрытие — его настройки.
- Некоторые пункты (Enchant Helper, Brew Helper, Messenger) — это **кнопки-
  открывалки меню**: нажал → открылось окно → сам модуль погас. Так и задумано.

---

## 3. Команды `/kafka`

Все команды клиентские (набираешь в чат, на сервер они не уходят).

| Команда | Что делает |
|---|---|
| `/kafka enchant` | Открыть помощник зачарований (меню выбора чар и пресетов) |
| `/kafka enchant <пресет>` | Вывести в чат готовый план для топ-шмота (напр. `helmet`) |
| `/kafka brew` | Открыть помощник зельеварения (меню как креатив-поиск) |
| `/kafka brew <зелье> [splash/lingering/long/strong]` | План варки конкретного зелья в чат |
| `/kafka tpa` | Меню авто-принятия телепортов |
| `/kafka tpa add <ник>` | Разрешить авто-TPA от игрока |
| `/kafka tpa remove <ник>` | Убрать из списка |
| `/kafka tpa list` | Показать разрешённых |
| `/kafka pm` | Открыть мессенджер (Discord-стиль) |
| `/kafka pm search <текст>` | Поиск по истории личных сообщений |
| `/kafka pm history <ник>` | Показать переписку с игроком |
| `/kafka pm read` | Сбросить счётчик непрочитанных |
| `/kafka group create <имя>` | Создать группу в мессенджере |
| `/kafka group add <ник> <группа>` | Добавить игрока в группу |
| `/kafka team add/remove/list <ник>` | Управление списком друзей |

> Ники и пресеты подсказываются автодополнением (Tab).

## 4. Фичи по категориям

### ⚔ Combat / фарм

- **Auto Repair** — чинит инструмент с Mending, стоя у нужного блока. Сам
  докидывает опыт, при выходе XP ставит паузу (`опыт закончился — пауза`),
  шифтом не спамит. Автоматически берёт из хотбара/инвентаря сломанную вещь с
  Mending на замену. Останавливается на `всё починено`.
- **Auto Eat** — автоматически ест при голоде.
- **Auto Mine** — авто-копка руды, с HUD-счётчиком «осталось руды».
- **Auto Replant** — авто-пересадка урожая.
- **Enchant Helper** (`/kafka enchant`) — считает **самый дешёвый порядок**
  зачарования (как сайт iamcal enchant-order): выбираешь нужные чары, мод
  ловит конфликты и показывает по шагам, что с чем совмещать на наковальне.
  Пресеты топ-шмота (`Топ Шлем`, `Топ Меч` и т.д.) с отчётом каких книг не
  хватает.
- **Brew Helper** (`/kafka brew`) — порядок варки любого зелья, включая
  **splash/lingering**, продление и усиление. Поиск по иконкам как в креативе,
  тултипы при наведении, итог с длительностью и уровнем эффекта.

### 🎨 Render / интерфейс

- **Pinned Order (HUD)** — закреплённый на экране план зачарования/варки,
  обновляется вживую. Кнопка «Закрепить» в меню Enchant/Brew. HUD можно
  **таскать мышкой прямо с открытым чатом**.
- Прочие ESP/оверлеи (здоровье, броня, предметы) — включаются в ClickGui.

### 👥 Friends / чат

- **Clickable Chat** — кликаешь по нику в личке → подставляется ответ; кликаешь
  по команде в чате → выполняется.
- **Private Messages** — красивое оформление лички «пузырями», локальная
  история с поиском (`/kafka pm search`), счётчик непрочитанных в HUD.
- **Mod Radar** — отмечает в табе (значок **✦**) тех, у кого тоже стоит этот
  мод. Работает по скрытому «рукопожатию» между модами.
- **Friend Chat** — напиши `// текст` — сообщение уйдёт всем друзьям онлайн
  (через серверный `/msg`, но оформлено как друзья-чат).
- **Auto Teleport** — авто-принятие `/tpa` **только** от игроков из белого
  списка (`/kafka tpa add`). Чужие запросы игнорируются.
- **Messenger** (`/kafka pm`) — см. следующий раздел.

## 5. Мессенджер (сообщения мимо сервера)

Личка и группы идут через **отдельный релей-сервер**, а не через Minecraft —
на игровом сервере этих сообщений не видно вообще. Нужно один раз настроить
подключение к релею (у вашей компании он общий).

**Настройка (ClickGui → Friends → Messenger):**

| Настройка | Что вписать |
|---|---|
| Server URL | адрес релея, напр. `ws://163.123.180.81:10041` (по умолчанию уже стоит) |
| Key | секретный ключ вашей компании (спроси у владельца релея) |
| Fallback via /msg | вкл — если релей недоступен, личка уйдёт через серверный `/msg` |
| Chat Notify | показывать входящие ещё и в обычном чате |

**Как пользоваться:**
- `/kafka pm` — открывает окно в стиле Discord: слева список диалогов
  (со счётчиком непрочитанных), справа сообщения с аватаркой-головой и ником.
- Диалоги: `@ник` — личка, `#группа` — групповой чат.
- Создать группу: `/kafka group create команда`,
  добавить: `/kafka group add Стив команда`.
- В шапке окна виден статус подключения к релею (онлайн/оффлайн).

> Если у тебя релей недоступен, а у собеседника тоже стоит мод — личка
> автоматически уйдёт скрытым whisper'ом через `/msg` (это уже видно серверу).

## 6. Частые вопросы

- **Ничего не открывается по X** — переназначь клавишу в Options → Controls.
- **Enchant/Brew Helper «выключается сам»** — так и надо, это кнопка открытия
  меню, а не постоянный модуль.
- **В мессенджере «релей недоступен»** — проверь Server URL и Key, и что у
  владельца релей запущен.
- **Значка ✦ нет у друга в табе** — у него не включён Mod Radar или он ещё не
  прислал ни одного сообщения через мод.
