# Мессенджер Kafka-Utils — инструкция

Встроенный чат между владельцами мода. Сообщения идут через **отдельный
релей-сервер мимо Minecraft-сервера** — в игровом чате их не видно. Есть личка,
группы, поиск, статусы онлайн, «печатает…», звук, редактирование и удаление
сообщений.

---

## 1. Один раз настроить (2 поля)

1. В игре нажми **`X`** — откроется меню Kafka-Utils.
2. Слева выбери категорию **Friends**, найди модуль **Messenger**, раскрой
   настройки.
3. Впиши два значения:

| Поле | Что вписать |
|---|---|
| **Server URL** | `wss://chat.kafkinse-ads.xyz` |
| **Key** | секретный ключ (тебе даст владелец лично — в этот файл он не пишется) |

Остальные настройки можно оставить по умолчанию:

| Настройка | Что делает |
|---|---|
| **Fallback via /msg** | если релей недоступен, личка уйдёт через серверный `/msg` (только 1-на-1, не группы) |
| **Chat Notify** | показывать входящие ещё и в обычном чате |
| **Sound** | звук при входящем сообщении |

> Важно: адрес начинается с **`wss://`** и **без порта** в конце. Ключ вводи
> точь-в-точь как дал владелец (регистр и символы важны).

Проверка: закрой меню и набери в чат **`/kafka pm`**. В шапке окна должно быть
зелёное **● релей**, а в игровой чат придёт «§aподключено к релею». Если
**● оффлайн** — проверь Server URL и Key.

---

## 2. Открыть мессенджер

Команда **`/kafka pm`** — открывает окно в стиле Discord.

Окно состоит из двух частей:

**Слева:**
- Поле **поиск** (сверху) — ищет по всем перепискам.
- Список чатов: `@ник` — личка, `#группа` — группа.
  - **●** зелёный = собеседник онлайн, **○** серый = оффлайн.
  - **(3)** красным = столько непрочитанных.
- Внизу поле **«ник / группа»** и кнопки:
  - **ЛС** — открыть личку с этим ником;
  - **Группа** — создать группу с таким именем;
  - **+уч** — (когда открыта группа) добавить участника из поля.

**Справа:**
- Сверху имя чата и статус собеседника.
- Для групп — кнопки **Участники** (список состава) и **Выйти**.
- Сами сообщения: аватар-голова и ник над текстом.
- Снизу — поле ввода и **Отправить**.

---

## 3. Как переписываться

- **Начать печатать** можно сразу — курсор уже в поле ввода, кликать не нужно.
- **Отправить** — клавишей **Enter** (или кнопкой «Отправить»).
- **Листать историю** — колесом мыши вверх; вниз — обратно к новым.
- **Написать новому человеку** — впиши его ник в поле слева внизу → **ЛС**.
- **Собеседник видит «печатает…»**, пока ты набираешь.

### Редактировать / удалить своё сообщение
1. **Кликни по своему сообщению** в переписке.
2. Сверху появятся кнопки **✎ Ред.**, **✕ Удал.**, **Отмена**.
   - **✎ Ред.** — текст подставится в поле ввода, поправь и нажми **Сохранить**.
   - **✕ Удал.** — сообщение станет «[удалено]» у тебя и у собеседника.

> Правки/удаления видят те, кто в этот момент онлайн в мессенджере. В
> сохранённой истории остаётся исходный текст.

---

## 4. Группы

- **Создать:** впиши имя в поле слева → **Группа**. Или командой
  `/kafka group create ИмяГруппы`.
- **Добавить участника:** открой группу, впиши ник в поле слева → **+уч**. Или
  `/kafka group add Ник ИмяГруппы`.
- **Кто в группе:** кнопка **Участники** справа сверху.
- **Выйти:** кнопка **Выйти**.
- Когда кого-то добавляют — всем в группе приходит уведомление в чат.

> Группы работают **только через релей** (в отличие от лички, у них нет
> запасного пути через `/msg`).

---

## 5. Команды (можно вместо меню)

| Команда | Действие |
|---|---|
| `/kafka pm` | открыть окно мессенджера |
| `/kafka pm search текст` | поиск по истории лички в чат |
| `/kafka pm history Ник` | показать переписку с игроком |
| `/kafka pm read` | сбросить счётчик непрочитанных |
| `/kafka group create Имя` | создать группу |
| `/kafka group add Ник Имя` | добавить в группу |

---

## 6. HUD непрочитанных (по желанию)

Хочешь видеть счётчик непрочитанных прямо на экране — включи модуль
**Messenger Unread** (X → Render). Показывает, в каком чате сколько новых
сообщений. Твои собственные сообщения непрочитанными не считаются.

---

## 7. Если что-то не так

- **● оффлайн / не подключается** — проверь: Server URL ровно
  `wss://chat.kafkinse-ads.xyz` (с `wss://`, без порта), ключ правильный,
  модуль Messenger включён.
- **Личка не доходит, релей оффлайн** — если включён Fallback, ЛС уйдёт через
  `/msg` (нужно, чтобы у обоих был включён **Friend Chat** и вы добавлены в
  друзья). Группы так не работают.
- **Нет звука** — включи настройку **Sound** у модуля.
- **Не появляется новая группа/чат** — открой любой чат или пересоткрой окно,
  список обновится.

Приятного общения! ✉
